# AI-Based Travel Itinerary Generator

A professional multi-page website prototype for a travel technology startup. This project demonstrates modern web design practices, dashboard analytics, and AI-powered service presentations.

## 🚀 Live Demo
Ready to be deployed on **Vercel** or **GitHub Pages**.

## 🛠️ Features
- **Modern Dashboard**: Real-time analytics, user statistics, and travel insights.
- **AI Planning Engine**: Smart itinerary generation and budget optimization.
- **Reporting System**: Performance analytics and destination popularity reports.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile devices.
- **Professional UI/UX**: Built with Semantic HTML5 and modern CSS3 (Inter & Outfit typography).
